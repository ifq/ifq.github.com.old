终于爬完富士山了，开始很想爬 -> 山上的时候大吼以后再也不爬山了 -> 现在又有点想爬山了，当然不是富士山
{% img /rc/legacy/bd8532adb4323c114a36d603.jpg 800 %}
整装待发，以为自己很猛。

{% img /rc/legacy/e73f915860d4f9c69d820403.jpg 800 %}
开始穿了好多，还穿了雨衣，结果走了几步一头汗，还是脱成这样了。

{% img /rc/legacy/6032ae4b28ce39e883025c03.jpg 800 %}
边上边穿，最后成这样了。3250之前已经死过一回，现在是回光返照的样子了。别人借给我的雨衣，咋就觉得眼熟，原来是公安雨衣。

{% img /rc/legacy/feb3d66216d4afc5e6113a0f.jpg 800 %}
八合目上边的时候，日出了。

{% img /rc/legacy/08704990e7e13995a877a408.jpg 800 %}
太阳出来了，发现离顶上还有那么远，崩溃了。看着破山，还不如褒河旁边的山。。

{% img /rc/legacy/2854bf31fb847eb05fdf0e0a.jpg 800 %}
富士山上唯一的植被。

{% img /rc/legacy/7a9e7509cad5e6b72fddd409.jpg 800 %}
山顶火山口上谁插了只鼓棒。

{% img /rc/legacy/3aa5b01c98e2eb9386d6b609.jpg 800 %}
终于上来了。后边的小坑就是火山口了。坐在这里写了张卡片，总算缓过来了。

之前看了很多攻略什么的，最后还是中招了。以前没有晚上负重爬过山，中间要挂了一样，小汗一冒，小风一吹，小寒蝉一打，小肚子一闹，小头一晕，算是功德圆满了。结果比同行的女同志还要慢很多到山顶，惭愧啊惭愧，亏我每周还去锻炼。
不过在山顶晒了晒算是缓过来了。先山只用了3小时整。听说一般都是4小时下山，自我安慰下。下山全是很厚很松德沙子，小跑起来倒是腿上不费劲，就是废鞋，幸好我穿的一双破鞋，报废了也不心疼。有同志为上山买了新鞋，我感到很惋惜。
上山之前看攻略里有些带上拐杖之类，觉得没必要。爬到一半发现有的地方确实需要，但是有的地方要棒子又是累赘。所以这个就看个人习惯把。 如果是准备的木质的拐杖还可以烙些印章在上边，倒是蛮不错。不过想到以后回国可能也带不会去，所以没买。
去之前有人告诉我一定要保持身体干燥，不能湿了。我就想怎么保持干燥呢。最后我还是湿润了。在后来我就中招了。所以这个真的很重要啊！可我还是不知道怎么干燥。。。
下次去哪儿之前再研究吧。
