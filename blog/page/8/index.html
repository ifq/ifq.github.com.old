Chrome 尝鲜

早上一来发现 Chrome 可以下载了。Down 了个学习学习。 速度蛮快的，界面也很明快，意料之中以及集成了 Google Gear。 也许有很多体贴的细节，还没体会到。总体感觉是一个较轻量级浏览器。 Google 就是这样了，不贪大求全，只凸显自己可人的地方，也许会叫人爱不释手。 比如说今天同时发布的另外一个东东，其实我更感兴趣。 （Google 的最近很活跃哦）今天还同时发布了 Picasa 3 。 看介绍，有个很体贴的功能，Name Tag。 可能类似于 Facebook 里那个 Name Tag吧。 不过当然比那个高级点–有人脸识别的能力。这样照片管理更方便了。 一直以来我就在希望能有类似的管理方式，因为照片太多，自己又懒的全部 tag。 用模式识别之类的技术来自动tag，我想还是有更大的扩展空间的吧:) 上班了。。

富士山游记

终于爬完富士山了，开始很想爬 -> 山上的时候大吼以后再也不爬山了 -> 现在又有点想爬山了，当然不是富士山

整装待发，以为自己很猛。


开始穿了好多，还穿了雨衣，结果走了几步一头汗，还是脱成这样了。

边上边穿，最后成这样了。3250之前已经死过一回，现在是回光返照的样子了。别人借给我的雨衣，咋就觉得眼熟，原来是公安雨衣。

八合目上边的时候，日出了。

太阳出来了，发现离顶上还有那么远，崩溃了。看着破山，还不如褒河旁边的山。。


富士山上唯一的植被。

山顶火山口上谁插了只鼓棒。

终于上来了。后边的小坑就是火山口了。坐在这里写了张卡片，总算缓过来了。

之前看了很多攻略什么的，最后还是中招了。以前没有晚上负重爬过山，中间要挂了一样，小汗一冒，小风一吹，小寒蝉一打，小肚子一闹，小头一晕，算是功德圆满了。结果比同行的女同志还要慢很多到山顶，惭愧啊惭愧，亏我每周还去锻炼。
不过在山顶晒了晒算是缓过来了。先山只用了3小时整。听说一般都是4小时下山，自我安慰下。下山全是很厚很松德沙子，小跑起来倒是腿上不费劲，就是废鞋，幸好我穿的一双破鞋，报废了也不心疼。有同志为上山买了新鞋，我感到很惋惜。
上山之前看攻略里有些带上拐杖之类，觉得没必要。爬到一半发现有的地方确实需要，但是有的地方要棒子又是累赘。所以这个就看个人习惯把。 如果是准备的木质的拐杖还可以烙些印章在上边，倒是蛮不错。不过想到以后回国可能也带不会去，所以没买。
去之前有人告诉我一定要保持身体干燥，不能湿了。我就想怎么保持干燥呢。最后我还是湿润了。在后来我就中招了。所以这个真的很重要啊！可我还是不知道怎么干燥。。。
下次去哪儿之前再研究吧。

So Many Things to Do, Which One Should I Do First???

很久没有认真更新了。天气也热了，都是等到8点以后温度下来了才下班。
今天一觉居然睡到了2点，虽然我定的11点的闹钟。搞得没力气去健身房了，
明天去。。
总觉得有太多事可以做，却不知道先做哪一件事，搞得很踌躇，
于是就去放松下，来盘 DOTA ，然后就该睡觉了。

想加速提高日语，但是就是提不起速度。买的书基本上只有大便的时候看。

最近部门里来了个美国研究生实习，经常蹭过去练2句英语，发现还是差远了，
于是又很想提高英语，于是做的事情又多了一样。

项目方面，终于做点感兴趣的了。要把老的程序跑到新的硬件上，
可以多接触的实质性的东西了。不像前端时间做的什么 Snapshot ，ClipTransition
虽然也能有所体会，但是太上层的功能性开发，对于以后没有太大帮助。
接触硬件机会多了，又有了自己学点东西的欲望。于是看了看 Wind River ，
下了新的 VxWorks ，准备在我的老ARM板上跑起来。于是要做的事又多了一样。

有这么多事可以做，但是这些事都不是能够迅速转化为Money的投资。
于是想做点短线投资。几个月前见朋友在玩外汇，于是也想了解下。
个人感觉外汇比股票单纯些，而且没有地域时间限制，比较适合我。
模拟炒了下，还是蛮复杂。很费脑子的。玩的就是概率和一点点判断。
其中的判断有2方面，基本面分析和技术面分析。
基本面就是根据各国的金融，政治，石油，股票等等的变革来判断。
技术面就是分析走势图，总结规律，有点统计学上的意思。
个人更倾向于基本面。但是情况是在太复杂，信息量太大，根本联系不起来。
于是想找个工具，能把价格走势图和发生的历史时间放在一个图上的，
可以参考学习下。但是还没有找到。实在不行就自己做个把。。
觉得用Google API 应该不难实现。但是还是最好能找到个比较牛的。
于是想做的事有多了一样，不过这个是事情还是麻烦点，还是先多问问朋友。

找了个外汇公司，申请了点资料。打开了 Google Reader, 列出一堆还没看的消息。
翻开了日语书。打开了VxWorks 的 PDF。
然后。。



打开GG平台。。
来一把DOTA 先！
