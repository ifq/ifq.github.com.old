第五天了.

前一天最喜欢的一张照片忘贴了.
{% img /rc/legacy/3769357aad4ba7cc2f73b3f7.jpg 800 %}


第五天赶早起来要看日照金山. 梅里雪山确实很壮观,高耸寂静的样子让人产生一种敬畏之情.
是当地藏民心中的一座神山,每年都要饶一圈朝拜.

前一天晚上在旅馆就看了关于梅里雪山的纪录片,讲的是90年代日本登山队在梅里雪山全军覆没的事.
至今梅里雪山还是一座处女峰.

梅里雪山有五个峰,主峰卡瓦格博峰(今天又想起来了),其他还有五指峰(?)将军峰....
{% img /rc/legacy/657636faadf195a89e5146ab.jpg 800 %}

{% img /rc/legacy/b0983a01c479562f7bec2cbe.jpg 800 %}

眼看阳光就要在山上展开了, 一团乌云飘来,主峰全部被挡住了.看来还真是要随缘啊.
昨天晚上的酒还是白敬了,还是没看全.
{% img /rc/legacy/df771e958336ec297bf48085.jpg 800 %}

{% img /rc/legacy/fb9c2d73f2b55d3d8701b091.jpg 800 %}


等了一会,看着没戏看到金山全貌了,只得继续赶路.下一站明永冰川.累坏了~
可以骑马上去,不过还是决定自己爬.(认识的都看出来了,其实是驴子..)
{% img /rc/legacy/d3a8f1d3d88efffea8ec9a52.jpg 800 %}

{% img /rc/legacy/6071b9a1e0869caa4610644e.jpg 800 %}

这冰确实感觉有好多年了.看着照片是挺安静的,其实是轰隆轰隆的,10几米的冰下面其实是流水.
这样掉下去就爽掉了.

刚喊了几句,就看从山顶处掉下一块冰,虽然不是很大,但是声音却很震撼.
要是这所有的冰的砸下来,那确实挺恐怖的. 想想梅里雪山突然消失的登山队....
