转_农民对于猪肉问题的讨论

猪肉涨价了才发现,猪肉是个男人都很关心的问题.


    无意间看了条新闻，一些城市人说猪肉涨价已经影响到了他们的生活。说实话，我还没有这样的感觉。我觉得他们说这些话有些骄情，肉那必竟不是馒头、白面，一顿不吃心害慌。所以初听到猪肉涨价，我的第一反应就是高兴，甚至兴高采烈。

也该涨价了！前 两年，随着玉米价格的不断上涨，农民养猪的效益也因之下滑，一些农民开始退出这个行当。或许这是这次猪肉价格突涨的主要原因，而猪肉给国人全国性的突然发 难，却是所有人始料未及的。而猪肉价格的暴涨也惊动了高层，总理确实是体恤民情，专程前往陕西调研猪肉生产供应的情况。看了新闻，作为陕西人我真感到荣幸 和高兴，总理也随之做了七点指示。记得总理来陕西还是三年前渭河的那次大水，那次是为了水，这次是为了什么？当然是为了老百姓的肉盘子。看了总理的七点指 示，总觉得有些缺憾，只有有一条是关于农民的，那就是要对饲养母猪的给予适当的补贴。尽管只有一条，我都觉得好感动。必竟有人想起那些苦苦挣扎的养殖户 了。尽管这条补贴在实施起来困难重重，也不会对时下疯涨的猪肉价格起到丝毫作用，但它必竟让养猪的农户心中泛起了丝丝甜意。试想，有多少养猪户不是在这暴 涨暴跌的轮回中倾家荡产，暴涨、暴跌的轮回始终都是众多养殖户的摆脱不了心悸。前多年，几乎每三年左右就一个轮回，没有人问过，也没有人想过那些在一次次 轮回中受伤的农民。所 以我感动，感动之后是我深深的担忧，这会不会又是一场风？其实这些问题，严格意义上说是市场中的问题，本该由那个看不见的手来解决，但我们似乎更习惯于看 领导，领导亲民是无错的，但会使那个市场的手消于无形，而这才是最可怕的。如何习惯用市场的手段解决市场中的问题，才是我们要习惯的。不过我们的有关部门 面对暴涨，学会了用市场的手段解决，那就是想到了平抑，云云要动用贮备库猪肉。似乎他们是天蓬大元帅猪哥哥，来个变变，变得肉肉无限。但时至今日，也不见 猪肉价格下跌，不知是库里没有还是说说大话吓市场，只是好长时间再也不见有关部门的声音了。所以，市场经济不是说说大话，就如股市一样，跌的时候政府再说 也不见涨，而涨的时候政府稍一开口就一泄千里。这个也怕是股民为什么不关心股却关心猪肉价格的原因吧。而这更是掀起了学界、媒体的关注，一时间猪猪满天 飞，而五大三粗的屠夫也在镁光灯的闪烁下频频亮相各级政府的电视新闻之中。经济学家关心暴涨的猪肉价格会不会引发通货膨胀，社会学家担心暴涨的猪肉会不会 加剧贫富阶层的对立，而各级领导想的是如何做出样子，来彰显爱民形象，而惶惶不安的股民则把猪看成了股，认为猪涨股也涨。但就是没有人想到养猪户，因为他 们都认为农民可发了，但他们可能不知道，这是农民在付出惨重代价后才显现的不正常现象。他们可能得到了些，但暴涨后引发的全民皆养可能就是他们的下一轮梦 魇。因为各级领导肯定会在总理要切实抓好生猪生产的指示下大抓特抓，出台许多刺激养猪的“好政策”、“好办法”，一哄而上，全哄而上，大干快干的养猪事业 轰轰烈烈的就开始了。我估计，不出两年就是猪肉价格的大跳水，而首先跳下深水的就是那最容易受伤的农民了。所以，我真的不愿看到总理那样的劳心费神，这 样，他真的是太累了。

我留意了各方给 出猪肉价格暴涨的原因。不外有两条：一是受疫情影响，存栏降低，导致市场供应减少；二是饲料价格上涨，养殖成本提高，农民养殖的积极性不高。归根结底一个 原因：供给不足。而为什么会突然上涨且无先兆？为什么会席卷全国？显然这个答案是不确切的，更是有些泛市场化，谁都知道市场本身就是由供求组成的，所以说 供应不足这个答案地球人都知道。其实这次猪肉暴涨，更多的暴露了我们社会中长期存在的深层次的问题。突然上涨，说明市场中出现了突发事件，而只有突发性的 事件才会导致市场的供给突然断档。是什么突发事件呢？只有大面积的疫病！但在我们的生活当中，并没有感到过这种事件的发生。其实在去年，南方就发生了大规 模的疫情，但没有人知道。由于前年年低市场猪肉价格的低迷，一些养殖户自然就做出了缩小养殖规模决策。而由于南方疫情造成的市场供给缺少已悄然形成，但由 于没有这方面的信息，在天灾和人为因素的相互作用下，猪一夜间悄然不见了，暴涨自然起风了。作为疫情农民应该不应该早知道？应该在多大的范围内发布？我查 了一下资料，好象在今年的六月初，新华社才发布了来自农业部的消息，去年南方爆发的高致病性猪蓝耳病给农民造成了较大的经济损失，但并未说明发病死亡的有 多少。看来我们的信息披露不但严重致后，而且极不准确。试想一下，如果农民早早得到这些信息，做出扩大养殖规模的决定，是不是会大大的缓解这一涨价风暴的 到来。这也同时反映了一个隐性的问题，如何给农民提供一个获得真实市场信息的环境。现在可以说是没有一个系统一个平台能真实、准确、快捷的给农民提供可用 的信息，农民获取信息的渠道可以说是五花八门，相当散乱，有报纸的、电视台的、也有网络的，但好象更多的是从广告、熟人那里获取。比如你现在要调查全国的 生猪数量，有农业部门的，有多经部门的，有统计部门的，有产业结构调整部门的，可以说是数出多门，但你无法确定那个数据真实。试想，这又如何让农民获取信 息呢，所以尽快整合各类信息，打造一个权威的农业信息系统和平台，让农民及时撑握第一手资料，获取有用的信息。

而为什么会席卷 全国呢？说明不是点上的问题，而是面上的问题，从全国性的猪慌来看，出现了全局性的问题。而我们又如何看待这个问题?这说明农业在整个国民经济发展中遇到 了新问题。什么新问题呢？那就是农业的比较利益进一步下降，与工业品的反差进一步拉大，其脆弱的行业本质在现代市场中越来越明显。我们还没有真正建立起来 一条顺畅的农产品供给链，市场好时就忘了农民，市场紧时就想起了农民。如这次猪肉风暴，如此的让领导们上心，主要原由还是城市的低收入者的呼声。而前年的 猪肉价格下跌，又有谁想到了农民？缺失了农民的和谐社会是偏面的和谐，是不长久的和谐，只能进一步拉大二者之间的差距。这又牵出了一个问题，那就是农民的 涣散，农民的没有组织性。如果农民组成有效的协会，就会通过协会传达呼声，也能够通过协会呼吁政府关怀。

涨价了是好事。 农民用农产品的涨价来对应农业比较效益下降，你不涨价，农民就选择离开，众多农民说的最多的就是养猪的效益下降，一头猪一百元的利润，真不如去打工。农民 终于在残酷市场的不断打击下清醒了，那就是选择了离开。离开土地，粮食涨价了，离开猪场了，猪肉涨价了。其实这是好事，随着农民不断的离开，是不是城市化 进程就会大大加快？或许吧，但引发的农村破败会不会出现？

我问一个养猪 户，涨价了，现在要多养呀。他是一个有着多年养猪经历的老户了。他说由于猪病，他把十几头母猪卖了，谁知猪涨价了。现在想在村外建个小区，上规模，但就是 没有猪可养。没有那么多的母猪，也一时买不到那么多的仔猪。是的，我国历来是养猪大国，但对于突然没有猪可养还是个有趣的问题。而到底如何养猪呢？美国的 养猪方法倒时值得我们学习。首先是遍及全美国的养猪协会，协会主要推广新品种，并协调有关政策，组织各种技术培训，更重要的是能给养猪户提供可靠的信息。 其次是政府建立了一套严格的种猪培育体系，将各个种猪场的信息进行登记，育种值进行汇编统一上网，便于农民查找，并进行信息共享；另外他们有很好的防疫体 系，这是我们的短处。这其实说明猪不光是农民在养，更是政府在养。政府要做的就是为农民提供一个好的养猪环境，让农民安心的养猪。使农民既养猪有利，城乡 居民又不因猪肉价格太高而远离肉香。若这样，也不至于累得我们的总理为难，不涨价，农民收入不增加，最让总理睡不安的就是农民的增收问题；涨价吧，又引发 物价飞涨，加居城市低收入者的生活困难，手心手背都是肉呀！

要回老家了，回去买些什物，还是割点肉吧。果真回家时老娘说，肉贵得不忍心割，几十天都没有问到肉味了，唉！好贵的肉、好香的肉呀！

查看 Ubuntu 版本二法

方法一

在终端中执行下列指令：

cat /etc/issue

可以查看当前正在运行的 Ubuntu 的版本号。其输出结果类似下面的内容：

Ubuntu 7.04 \n \l

方法二

使用 lsb_release 命令也可以查看 Ubuntu 的版本号，与方法一相比，内容更为详细。执行指令如下：

sudo lsb_release -a

将输出结果：

Distributor ID: Ubuntu
Description: Ubuntu 7.04
Release: 7.04
Codename: feisty

理解相对时间

最近空闲时间重温了霍金的时间简史,比起高中时看的时候多了些思考.但是时间的解释还是很模糊,于是在网上找到这篇文章,帮助很大.

相对时间、相对空间、相对物质

赵丰军

今年是爱因斯坦提出相对论100周年的时间，世界物理学界认为这是一个有意义的年份，特别确定2005年为物理年，目的之一是为了纪念著名物理学家爱因斯坦，同时也是为了寻找另一位新时代的爱因斯坦，发展物理家，重振物理学以前的雄风，让物理学为当今世界人民做出更大的贡献；

为物理年增光增辉，特别写这一篇文章，用光子是物质的基本粒子这一理论，重新解读爱因斯坦的相对论，并且增加“物质相对”这一内容。

一、相对时间　 我们知道爱因斯坦为了让世人理解“光电效应”，在普朗克的物理理论基础上，提出光量子的理论，简称为光子，又在光子的传播速度为自然界的最大速度这个前提 下，提出了“相对论”，这个理论改变了人们长期思想认识中的绝对时间、绝对空间概念，从而发展了人们认识中的宏观物理学，成为当今量子物理科学的奠基人之 一。

我 们学习了牛顿力学之后，基本上知道自然界的物理规律，但是人们心中有一个认识，时间总是向前发展的，空间是多少就是多少，也就是人们常说的谁也不能改变时 间，不能让时间增多，不能让时间减少，不能让时间倒退，就是说人们不能改变时间进程，也可以用通俗的话讲，人们不能改变时间，时间是一个绝对时间，但是在 爱因斯坦的相对论中，时间是一个可以变化的物理量，可以人为改变，让一个钟表，用接近于光速的速度运行，这个时钟会相对于不运动的时钟变慢，从这个意义上 讲，人们改变了时间进程，时间不是一个绝对的物理量，而是一个相对的物理量，时间是一个相对的时间，这是爱因斯坦的相对论改变了人们的观念。但是时间是一 个相对性的概念，让更多的人不理解，说爱因斯坦的相对论是错误的，反对相对论，这里从物质的基本粒子说一说时间的相对性，看一看人们能不能从这个角度理解 时间的相对性。

人 类寻找物质的基本粒子，已经有几千年的历史了，可以认为分子的提出、中国的金木水火土就是人类寻找物质的基本粒子的开始，但是基本粒子与科学家周旋的时间 太长，人们找到一个质量比较小的基本粒子，结果又会发现一个比原来质量更小的基本粒子，让科学家们一次次存在希望，结果又是一次次失望，目前物理学界，科 学家们的心中不再相信有基本粒子这个概念。人们应该换一个角度思考问题，既然找一个质量小的基本粒子，还会有更小质量的基本粒子来代替，我们为什么不用没 有静止质量的粒子充当基本粒子呢，它就是爱因斯坦的提出的“光子”。当然我们说光子是物质的基本粒子，还有其它几个重要原因，详情可访问:

　 用光子这种粒子组成的物质，具有不同的特征，物质存在的时候没有静止质量，我们观察到的静止是质量，是物质不断与环境中的其它物质，相互作用光子或光子信 息，表现出的质量，我们人类长期以来认为它就是物质的固有质量，这是一个错觉。在这个理念下，时间是相对的，当一个物体与环境中的其它物质相互作用交换光 子能量的时候，自己的光子组成才能发生变化，自己的生命才能向前移动，时间才能有意义，如果物质根本就不能与环境中的其它物质相互作用光子信息能量，自己 的光子信息组成就不发生变化，自己的生命历程没有前进，其时间对它来说就没有意义，它身边的时钟根本就不会走动。这种事例非常少，但是在我们地球上确实发 生过。一个人失踪几十年后回来，没有变老，从雷达上失踪的飞机，乘客的手表和地面上的表相比较变慢，都是这种实例。从这个角度来看，在生活中时间也是相 对，只有对待有生命历程的物体，时间才有意义。

二、相对空间　生 活阅历告诉我们，一个物体是多长就是多长，这样就会形成绝对空间的概念，无论是牛顿以前，还是牛顿之后，还是今天的人们思想中，人们仍然坚持生活中的认识 ――――绝对空间，物体的长度不能改变，但是在爱因斯坦的相对论中，如果物体以某一个速度运动，在运动方向上物体的长度会变短，这是一个反传统的理念，世 人不理解，更多的人反对相对论。

我们人类是一种有思想、有文化的科学人，绝不能因我的感知而存在，应该相信一定的逻辑思维过程，相信我们没有见过，没有感知到，但又是存在的物理过程。

我们知道，一个直径是1米的球形物体，在我们的生活阅历中有多大，它就是1米， 如果有人说这个物体直径有几万光年，你也不要轻易反对，因为他说的直径是几万光年是另一种空间，是在某种情形下的空间，是相对空间，我们静止地看这个物体 是一米的空间，是我们生活中的实例，但是当这个物体处于一种特殊情形下，直径会变为几万光年。当你听完这一句话的时候，一定不相信这是真的，下面一个实例 就是这种情况。

我们假定物质的基本粒子就是光子，只有这个直径是1米 的金属球中的光子与环境的光子相互作用之后，金属球才能将自己的质量表现给自然界，能不能相互作用，就是看金属球中的光子与环境的光子信息是不是相差太 远，如果相差太远，金属球中的光子信息就不能与环境中的光子信息相互作用，金属球的光子信息就不能变化，金属球就不能将自己的所有特征表达给自然界，自己 也就不发生变化，在我们人类看来，金属球会突然消失，像是科学家们说的金属球进入了另一个时空，实事上没有，金属球就在原地，只是它不能与环境作用光子信 息能量，不能表现自己。但是，对金属球内部的所有物质还是正常的，光子仍然以光速度运动，但是对金属球中的光子来讲，这个光子永远也不离开金属球，也就是 说光子从金属球的边缘出发，永远也到达不了另一边，这就是金属球的直径是几万光年的原因。

从上面的实例可以看出，我们所说的空间也是相对的，它正是爱因斯坦理论的实际描述，只是爱因斯坦不知道光子就是基本粒子，没有将相对论与实际相结合。

三、相对物质　说空间相对，说时间相对，人们还是可以接受的，但是如果说物质相对，会有更多的人反对这个观点。所谓的物质相对可以理解为现在是这种物质，过一会它可能转变成其它物质；物体现在的质量是100千克，过一会它的质量可能变成200千克。由于生活中没有这种现象，相信的人非常少。

由 于光子是物质的基本粒子，光子没有静止质量，光子构成物质本身，对环境中的其它物体，对观察者来说没有意义，只有它与环境作用的这一部分光子信息才有意 义，单位时间内这个物体与环境作用的光子信息能量多，它显示的质量就大，单位时间内这个物体与环境作用的光子信息能量少，它显示的质量就小。所以物体的质 量大小，并不是取决与构成物质本身，而是取决它与环境作用的光子信息能量。所以物质的质量是可以随时改变的，事实上生活中的物质质量确实在不断改变，处于 不同的环境光子信息中，它与环境作用的光子信息能量不同，它表现出的质量就是不同的，但是这个改变量非常小，没有引起人们的关注。

关 于物质的形状、形态，物质种类也是在不断发生着变化，只是变化的程度没有影响人们的生活，人们说是量变，没有发生质变。从这个意义上，物质是在不断变化， 物质的存在价值不是取决于自己，而是取决于自己与环境共同结合后的结果，所以讲，物质也是相对的，不同的环境表现出不同的物质，物质也是相对的。

　这就是从物质基本粒子的角度，解释了爱因斯坦的相对论。我们周围的一切都是相对的。
